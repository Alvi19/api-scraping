# Cara Menjalankan Project Express

## 1. Clone repository
```bash
git clone <https://github.com/Alvi19/api-scraping.git>
cd <api-scraping>
```

## 2. Install dependencies
```bash
npm install
```
## 3. Jalankan Api service
```bash
npm start
```

## 4. Jalankan testing (stress test)
```bash
node test-stress
```
